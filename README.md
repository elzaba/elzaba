## 🚀 Projects

### 🏷️ PricePing — E-Commerce Price Tracker

[![Live](https://img.shields.io/badge/🌐_Live_at-priceping.fly.dev-6C63FF?style=flat-square)](https://priceping.fly.dev)

Track product prices across e-commerce sites and get email alerts when they drop below your target.

Built with a two-tier scraping pipeline — `httpx` for standard pages with a **Playwright/Chromium** fallback for bot-protected sites. FastAPI async backend, APScheduler for automated checks, JWT auth with bcrypt and rate limiting, GitHub Actions CI across Python 3.11/3.12/3.13, and deployed on Fly.io with persistent volumes.

Developed with **Claude Code** as a pair programming tool — a good experiment in AI-assisted development and knowing when to override the suggestions.

`Python` `FastAPI` `SQLite` `Playwright` `Docker` `GitHub Actions` `Fly.io`
